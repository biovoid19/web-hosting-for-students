    Detalles Técnicos Explicados

    Creemos que enseñar sobre hosting es una parte importante de la educación web. A continuación, encontrarás algunas explicaciones de los componentes mayores que hacen una cuenta de web hosting así como información de cómo hemos configurado para nuestras cuentas.

    Almacenamiento

    Esta medida determina que tanto almacenamiento de archivos tienes en tu cuenta. En el mundo del hosting, este es normalmente uno de los componentes más caros de hosting. Afortunadamente, desde que los datos y archivos son optimizados (comprimidos) para la web, incluso los sitios grandes raras veces ocupan mucho espacio de almacenamiento. la mayoría de los estudiantes ocupan menos de 0.25GB.

    Ancho de Banda

    Esta medida relaciona el número de visitas que tu o tus sitios pueden soportar y qué tanto contenido puede descargar o subir la gente desde tu sitio. Con 10GB de ancho de banda por ejemplo, tus proyectos pueden obtener miles de visitas sin problema alguno. Si de repente te vuelves super famoso o viral, no te preocupes, te podemos ayudar con el ancho de banda que necesites.

    Bases de Datos

    Si utilizas una Aplicación de Sistema de Gestión de Contenidos (CMS), es probable que requieras de una base de datos para almacenar tus datos. Las bases de datos te permiten almacenar de forma eficaz los datos y acceder a ellos desde tu aplicación o sitio web. Nuestras cuentas vienen con MySQL SQLite, PosgreSQL. también contamos con phpMyAdmin, la herramienta popular para ver y editar tus bases de datos

    Software de cPanel Hosting

    Una gran parte de las cuentas de hosting de ahora utilizan el software de cPanel para la gestión de configuraciones. Amamos ofrecer cPanel porque es fácil de aprender y también es una habilidad que los estudiantes probablemente la utilicen en un futuro.

    FTP, SFTP, SSH, Git

    Una gran parte de las cuentas de hosting de ahora utilizan el software de cPanel para la gestión de configuraciones. Amamos ofrecer cPanel porque es fácil de aprender y también es una habilidad que los estudiantes probablemente la utilicen en un futuro.

    Dominios Addon

    Este es el término técnico de cómo permitimos que algunas de nuestras cuentas hosting múltiples nombres de dominios en una sola cuenta. Técnicamente cada nombre de dominio tiene su propia carpeta en el servidor dentro de public_html. Los visitantes sin embargo, ven cada sitio con su propia URL

    Email Hosting

    La mejor práctica para hostear correos electrónicos es de no hostear tu correo electrónico en el mismo servidor. Esto garantizará que los problemas que surjan en tu sitio no afectarán a tu correo electrónico. También, la cantidad de espacio que tienes para correo viene del total de espacio de almacenamiento. Mientras puedas alojar tu correo con tu cuenta de estudiante, te recomendamos servicios de terceros como Zoho Mail y Google Apps.

    Sistema de Gestión de Contenido

    Cuando te metes a crear sitios web, te encontrarás con un sistema de gestión de contenidos (CMS) como WordPress, Drupal, Joomla u otras más. No solo soportamos la mayoría de los CMS, también tenemos instalaciones de un solo click disponibles. Esto permite a instalar de forma rápida y fácil CMS sin la necesitas de saber mucho de bases de datos o programación.

    Especialista en WordPress

    WordPress es uno de los CMS más populares y uno al que amamos dar soporte. Nuestro personal de soporte sabe mucho de WordPress y seguidamente pueden guiarte a los mejores recursos para resolver problemas relacionado con WordPress. También ofrecemos auto-instalador de 1 click de WordPress.

    Soporte de Lenguaje Moderno

    Un certificado SSL, certifica y encripta el contenido de tu sitio, permitiéndote correr cosas como sitios de e-commerce. Ahora estamos ofreciendo certificados SSL de forma gratuita para todos los estudiantes utilizando la Autoridad de Certificados Encriptados.

    Certificados SSL

    An SSL, or secure sockets layer, certificate encrypts the content on your site, allowing you to run things like ecommerce sites. We are now offering SSL certificates for free to all students using the Let’s Encrypt Certificate Authority.

    RAM Dedicada

    La RAM es la memoria que utiliza tu servidor para procesar muchas tareas involucradas con la configuración y funcionamiento del sitio. Tenemos configuraciones especiales en nuestros servidores para asegurarnos que otra RAM de estudiante no alente tu sitio.
